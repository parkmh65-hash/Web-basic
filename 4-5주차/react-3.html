CSS object-fit 속성 가이드

이미지가 컨테이너 크기에 어떻게 맞춰지는지 확인해보세요.

object-fit: fill

비율을 무시하고 컨테이너를 가득 채웁니다. (이미지가 왜곡될 수 있음)

object-fit: contain

비율을 유지하며 이미지가 전체가 보이도록 맞춥니다. 여백이 생길 수 있습니다.

object-fit: cover

비율을 유지하며 컨테이너를 가득 채웁니다. 이미지의 일부가 잘릴 수 있습니다.

object-fit: none

이미지 크기를 조절하지 않고 원본 크기 그대로 보여줍니다.

object-fit: scale-down

none과 contain 중 더 작은 크기로 이미지 사이즈를 조절합니다.
